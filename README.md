# 多功能小键盘

## 板载硬件

### 蓝牙模块

通过蓝牙模块与MCU和电脑建立蓝牙连接，提供无线使用功能。

### MCU

HK航顺芯片，因为本款MCU没有USB引脚支持，因此采用的是串口发送数据，需要板载USB转TTL。

### 充电电路

对无线提供电源功能。

### 旋钮EC11

通过EC11模块可以执行旋转，按下旋转等功能。

### 按键

提供四个机械按键、四个静音按键和一个侧边按键。

## 软件编程

### python上位机

通过python上位机对按键功能进行重定义。

##  其他功能

可将其接入智能家居类，控制智能家具亮度等。
通过按钮绑定MQTT消息即可实现。
